### ⚡ **Lightning Setup**

<table>
<tr>
<td width="50%">

**1️⃣ Clone & Install**
```bash
git clone https://github.com/YuvrajsinhRaj/AI-Fitness-assistant.git
cd AI-Fitness-assistant
npm install
```

**2️⃣ Environment Setup**
```bash
cp .env.example .env.local
# Add your API keys ⚡
```

</td>
<td width="50%">

**3️⃣ Database Init**
```bash
npx convex dev
# Real-time database ready! 🚀
```

**4️⃣ Launch**
```bash
npm run dev
# Visit http://localhost:3000 🎉
```
